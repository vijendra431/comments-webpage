# 📝 Comments Web App (React)

This is a **React-based Comments Application** where users can add, like, delete, and search comments.  
It is built using **React Class Components** and demonstrates important concepts like state management, event handling, and list rendering.

---

## 🚀 Features

### ✅ Add Comment
- User can enter name and comment
- New comment is added dynamically

### ✅ Input Validation
- Prevents empty name or comment
- Limits comment length to 200 characters

### ✅ Random Background Color
- Each comment gets a random color
- Improves UI appearance

### ✅ Like Toggle ❤️
- Users can like/unlike comments
- State updates dynamically

### ✅ Delete Comment 🗑
- Users can delete any comment
- UI updates instantly

### ✅ Search Comments 🔍
- Search comments by name
- Case-insensitive filtering

### ✅ Comments Count
- Displays total number of comments

### ✅ Latest Comments First
- New comments appear at the top

---

## 🛠 Technologies Used

- React JS (Class Components)
- JavaScript (ES6)
- CSS
- UUID (for unique IDs)
